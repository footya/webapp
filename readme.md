#度美食前端
>前端基础angularjs + requirejs + bootstrap, 以[Mobile Angular UI](http://mobileangularui.com/) 为主要参照

##项目目录结构

```
dist      -- 项目压缩文件，供上线使用
src       -- 项目源码，开发主要在这个目录下进行
tests     -- 测试用例
tools     -- 构建工具等
```
